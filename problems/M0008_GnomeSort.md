## 题目描述

问题：
```
地精排序Gnome Sort
```

## 思路
- 只有一层循环，循环过程进行回退，排序过程从左到右，一旦遇到交换的情况发生就往回退一步再比较。

- [其他思路](https://www.cnblogs.com/kkun/archive/2011/11/23/2260296.html)

## 关键点
- 回退
- 交换
