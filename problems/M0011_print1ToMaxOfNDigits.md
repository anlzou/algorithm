## 题目描述

问题：
```
输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。
```
示例：
```
比如输入 3，则打印出 1、2、3 一直到最大的 3 位数即 999。
```

## 思路
由于 n 可能会非常大，因此不能直接用 int 表示数字，而是用 char 数组进行存储。
