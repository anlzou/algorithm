## 题目描述

问题：
```
n个元素中取m个元素的组合
条件 m <= n
```
示例：
```
数组 a = [1,2,3,4,5,6]
选择 m = 3
```
结果：
```
$01 [1,2,3]		$02 [1,2,4]		$03 [1,2,5]		$04 [1,2,6]		$05 [1,3,4]
$06 [1,3,5]		$07 [1,3,6]		$08 [1,4,5]		$09 [1,4,6]		$10 [1,5,6]
$11 [2,3,4]		$12 [2,3,5]		$13 [2,3,6]		$14 [2,4,5]		$15 [2,4,6]
$16 [2,5,6]		$17 [3,4,5]		$18 [3,4,6]		$19 [3,5,6]		$20 [4,5,6]
```
